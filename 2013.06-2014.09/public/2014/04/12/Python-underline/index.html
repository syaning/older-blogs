Python中的下划线

在python编程中，经常会遇到以下划线开头的属性或方法，常见的有以下四种情况：

  • xxx，不以下划线开头，表示public
  • _xxx，以单个下划线开头，表示private，只有类对象可访问，子类对象不可访问，不可以通过from module import *导入
  • __xxx，以双下划线开头，类似于protected，类对象及子类对象可访问
  • xxx，以双下划线开头和结尾，是系统定义的一些特殊名称
